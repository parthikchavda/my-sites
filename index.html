FAQ

Frequently asked questions

Everything you need to know about this dashboard.

This is a personal site manager that pulls all of your Netlify-deployed websites using the Netlify API. It gives you a searchable, filterable bird's-eye view of everything you've published — sorted by category, viewable in grid or list layout.
Yes. Your personal access token is stored exclusively in your browser's localStorage. It is never transmitted to any third-party server — all API calls go directly from your browser to the official Netlify API at api.netlify.com.
Log into Netlify, go to User Settings → Applications → Personal access tokens and click "New access token". Give it a descriptive name and copy the generated token. Paste it into this dashboard and click Connect.
Site data is cached in localStorage for 5 minutes to keep the dashboard fast and reduce unnecessary API calls. If you've just deployed a new site and don't see it, wait a few minutes and reload — or reset your token to force a fresh fetch.
Categories are detected automatically by matching keywords in the site name and URL against patterns for subjects like Mathematics, Electronics, Physics, Communications, and more. Sites that don't match any pattern are labelled "General".
Absolutely. The dashboard is fully responsive. On small screens the nav collapses into a hamburger menu, the token entry form stacks vertically, and the site grid switches to a single column for comfortable browsing.
